# tm-contract-vary

마스터 양식 하나에서 **이름·조건만 바꿔** 계약서를 PDF로 뽑고, 파일로 버전 관리하는 도구.
1인 사업가·프리랜서의 반복되는 계약서/견적서 작업을 없애줍니다. 코딩 몰라도 VS Code에서
Claude Code에게 말로 시키면 됩니다.

- 👉 **시작하기:** [`0.먼저읽기.md`](0.먼저읽기.md)
- 🛠️ **직접 만들어보기(실습):** [`실습워크시트.md`](실습워크시트.md)

## 어떻게 도나요

```
1.신청서넣기  📥 구글폼 응답(CSV)·메모를 넣는 곳
2.계약서양식  📄 계약서 틀(양식) — {{변수}}만 빈칸
3.내정보      🙋 나·거래처 정보 (한 번만 채우면 재사용)
4.완성계약서  ✅ 완성된 계약서가 HTML·PDF로 나옴
```

Claude Code 스킬(`_스킬/contract-vary`)로 동작합니다.
폴더에 든 예시는 **광고 대행 용역 계약서**이고, 데이터는 전부 **가상 더미**(홍길동·㈜밝은무역)예요.
실습은 이 예시를 **내 계약서로 갈아끼우는 것**이 목표입니다.

## 자동화 레벨

- **L1 — 로컬 파일** (이 레포): 양식 + 값 → HTML·PDF, 파일이 곧 버전 이력
- **L2 — 구글폼 → 시트에서 한 번에** 일괄 생성
- **L3 — 생성된 PDF 메일 자동 발송** (MCP 연동)
